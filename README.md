# 災害AR PWA（GitHub Pages版・避難ルート矢印つき）

この版では次を追加しています。

- 避難所マーカーから「ここへ避難」で目的地選択
- 地図上に徒歩ルート表示
- AR画面で次の進行方向を大きな矢印で表示
- 既存の投稿保存、JSON/GeoJSON出力、PWA対応

## 使い方

1. 地図を開く
2. 避難所マーカーをタップして「ここへ避難」
3. AR画面に移動
4. 大きな矢印に従って進む

## 注意

- ルート計算はオンラインで取得します
- ルートAPIが応答しない場合は、通常の地点AR表示だけ動きます
- 位置情報、カメラ、方位は HTTPS で開いてください
